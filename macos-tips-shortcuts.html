Trucos y Atajos de macOS: Optimiza tu Flujo de Trabajo

macOS está lleno de funcionalidades ocultas y atajos de teclado que pueden acelerar drásticamente tu productividad. Conocer y utilizar estos trucos y atajos te permitirá navegar, gestionar archivos y ejecutar comandos de forma más eficiente, haciendo que tu experiencia con Mac sea aún más fluida y poderosa.

Atajos de Teclado Esenciales

Estos son algunos de los atajos más útiles que todo usuario de macOS debería conocer:

  • Cmd + C / Cmd + V

    Copiar / Pegar.

  • Cmd + X

    Cortar (para texto). Para mover archivos en Finder, usa Cmd + C para copiar y luego Option + Cmd + V para mover.

  • Cmd + Z

    Deshacer la última acción.

  • Cmd + S

    Guardar el documento actual.

  • Cmd + F

    Abrir la barra de búsqueda en la aplicación actual o Finder.

  • Cmd + Q

    Cerrar la aplicación activa.

  • Cmd + W

    Cerrar la ventana actual.

  • Cmd + Tab

    Cambiar entre aplicaciones abiertas (mantén Cmd presionado y usa Tab para seleccionar).

  • Cmd + Espacio

    Abrir Spotlight Search (búsqueda rápida de archivos, aplicaciones, conversiones, etc.).

  • Cmd + Mayús + 3

    Tomar una captura de pantalla de toda la pantalla (guardada en el escritorio).

  • Cmd + Mayús + 4

    Tomar una captura de pantalla de una selección específica (arrastra para seleccionar un área).

  • Cmd + Mayús + 5

    Abrir las opciones de captura de pantalla (grabar pantalla, seleccionar ventana, etc.) (macOS Mojave y posterior).

  • Cmd + Control + Espacio

    Abrir el visor de caracteres para emojis y símbolos.

  • Control + Cmd + Barra Espaciadora

    Abrir el selector de Emojis y Símbolos (alternativa a la anterior en algunos contextos).

  • Cmd + Option + Esc

    Forzar la salida de una aplicación que no responde.

  • Cmd + , (coma)

    Abrir las preferencias de la aplicación activa (si está disponible).

  • Cmd + N

    Abrir una nueva ventana o documento.

  • Cmd + T

    Abrir una nueva pestaña en el navegador o Terminal.

  • F3 (o Fn + F3 en algunos teclados)

    Activar Mission Control para ver todas las ventanas abiertas y escritorios virtuales.

Trucos Útiles de macOS

  • Vista Rápida (Quick Look): Selecciona cualquier archivo en Finder y presiona la barra espaciadora para obtener una vista previa instantánea sin abrir la aplicación.
  • Stacks en el Escritorio: Organiza automáticamente los archivos de tu escritorio en grupos ordenados. Haz clic derecho en el escritorio y selecciona "Usar grupos" (Use Stacks).
  • Hot Corners (Esquinas Activas): Configura las esquinas de tu pantalla para activar acciones como Mission Control, Mostrar Escritorio, o poner la pantalla en reposo al pasar el ratón. Ve a Ajustes del Sistema > Escritorio y Dock > Esquinas activas.
  • Modo Oscuro: Reduce la fatiga visual activando el modo oscuro. Ve a Ajustes del Sistema > Apariencia > Oscuro.
  • Dictado: Presiona la tecla Fn dos veces (o configura un atajo en Preferencias del Sistema > Teclado > Dictado) para dictar texto en cualquier campo de entrada.
  • Ajustes de Batería (Portátiles): En portátiles, ve a Ajustes del Sistema > Batería para gestionar el estado de la batería, optimizar la carga y ajustar el rendimiento.
  • Renombrar Múltiples Archivos: Selecciona varios archivos en Finder, haz clic derecho y selecciona "Renombrar X ítems". Puedes añadir texto, reemplazarlo o formatear.
  • Firmar Documentos en Vista Previa: Abre un PDF o imagen en Vista Previa, haz clic en el icono de Marcado (Toolbar) y luego en el icono de Firma para añadir tu firma digitalizada (puedes crear una usando tu trackpad o la cámara).

Personalización de Atajos de Teclado

Puedes crear tus propios atajos de teclado para cualquier comando de menú en cualquier aplicación. Ve a Ajustes del Sistema > Teclado > Atajos de teclado. Aquí puedes añadir atajos personalizados para aplicaciones específicas o para todo el sistema.
