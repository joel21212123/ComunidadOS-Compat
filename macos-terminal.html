Terminal de macOS: Potencia la Gestión de tu Sistema

Aunque macOS es conocido por su interfaz gráfica intuitiva, debajo del capó se encuentra un potente sistema operativo basado en Unix, lo que significa que tienes acceso a una línea de comandos robusta a través de la aplicación Terminal. La Terminal de macOS, al igual que en Linux, te permite ejecutar comandos para realizar tareas avanzadas, automatizar flujos de trabajo, gestionar archivos y procesos, y acceder a funcionalidades que no están disponibles en la interfaz gráfica.

Accediendo a la Terminal

Puedes abrir la Terminal de varias maneras:

  • Ve a Aplicaciones > Utilidades > Terminal.
  • Usa Spotlight Search (Cmd + Espacio) y escribe "Terminal".

Comandos Básicos de la Terminal de macOS

Aquí tienes algunos comandos esenciales que te ayudarán a empezar:

Navegación de Archivos y Directorios

  • pwd 
    pwd

    Imprime el directorio de trabajo actual (Print Working Directory).

  • ls 
    ls

    Lista el contenido del directorio actual.

    ls -l

    Lista el contenido con formato largo (permisos, propietario, tamaño, fecha). 

    ls -a

    Muestra todos los archivos, incluyendo los ocultos.

  • cd 
    cd Documentos

    Cambia al directorio "Documentos".

    cd ..

    Sube un nivel en la jerarquía de directorios.

    cd ~

    Vuelve al directorio de inicio (home directory).

  • mkdir 
    mkdir MiNuevaCarpeta

    Crea un nuevo directorio llamado "MiNuevaCarpeta".

  • rmdir 
    rmdir MiCarpetaVacia

    Elimina un directorio vacío.

Gestión de Archivos

  • touch 
    touch nuevoarchivo.txt

    Crea un archivo vacío o actualiza su marca de tiempo.

  • cp 
    cp archivo.txt /ruta/a/destino/

    Copia `archivo.txt` a la ruta especificada.

  • mv 
    mv archivo_viejo.txt archivo_nuevo.txt

    Mueve (o renombra) un archivo.

  • rm 
    rm archivo.txt

    Elimina un archivo.

    rm -r CarpetaConContenido

    Elimina un directorio y su contenido (¡úsalas con precaución!).

  • cat 
    cat archivo.txt

    Muestra el contenido de un archivo.

Comandos del Sistema y Utilidades

  • sudo 
    sudo apt update

    Ejecuta un comando con privilegios de superusuario (requiere tu contraseña de administrador).

  • man 
    man ls

    Muestra el manual de un comando (presiona 'q' para salir).

  • pbcopy / pbpaste 
    echo "Este texto irá al portapapeles" | pbcopy

    Copia el resultado de un comando al portapapeles.

    pbpaste

    Pega el contenido del portapapeles en la terminal.

  • open 
    open .

    Abre el directorio actual en Finder.

    open miarchivo.txt

    Abre `miarchivo.txt` con su aplicación predeterminada.

  • top / htop 
    top

    Muestra los procesos en ejecución y el uso de recursos. 

    brew install htop

    `htop` es una alternativa más interactiva (requiere Homebrew).

  • ping 
    ping google.com

    Envía paquetes para verificar la conectividad a un host.

Personalizando la Terminal

Puedes personalizar la Terminal de macOS para mejorar tu productividad y experiencia:

  • Preferencias de Terminal: Desde el menú Terminal > Preferencias, puedes cambiar los colores, las fuentes, el fondo y las configuraciones de shell para crear perfiles personalizados.
  • Oh My Zsh: Una popular extensión para Zsh (el shell predeterminado en macOS desde Catalina) que proporciona temas, plugins y una mejor experiencia de línea de comandos.
  • Homebrew: El "gestor de paquetes que le falta a macOS". Permite instalar fácilmente herramientas de línea de comandos (ej., `htop`, `wget`, `tree`) y aplicaciones. 
    
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

    Comando para instalar Homebrew.

Consideraciones

  • Cuidado con `sudo` y `rm`: Utiliza `sudo` con precaución, ya que otorga privilegios de superusuario que pueden causar daños si se usan incorrectamente. El comando `rm -rf /` (borrar recursivamente la raíz del sistema) es un ejemplo de un comando peligroso que nunca deberías ejecutar a menos que sepas exactamente lo que haces.
  • Autocompletado: Presiona la tecla `Tab` para autocompletar nombres de archivos y comandos. Esto es muy útil y ayuda a evitar errores tipográficos.
  • Historial de Comandos: Usa las flechas `Arriba` y `Abajo` para navegar por los comandos ejecutados previamente.
